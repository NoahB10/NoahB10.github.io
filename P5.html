MATLAB Crane Game

Learned about the various aspects of how CAD softwares operate from creating objects, perspectives, projections, object transformations, shading methods and more.

Coded from skratch a Crane Game in MATLAB which has rules, levels, obstacles, lights and constrols.

o 3D game with hit box's and goals

o Custom made 7 level game or freeplay sandbox

Textures imported and wrapped on objects

o 3 vewing perpectives
    1. Cabin View - perspective as if driving from within the cabin
    2. Top View - 2D perspective of the game from above
    3. Flying Camera - custom viewing perspective by maneruvering the camera globally

o Modifyable settings and lighting
